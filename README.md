## Overview

This portfolio showcases the work and expertise of Julius-Attah Justice Chukwulaka, a Full-Stack Developer and DevOps Engineer based in Lagos, Nigeria. Built with a modern tech stack featuring React 19, Next.js 14, and TypeScript, it demonstrates proficiency in both frontend development and backend infrastructure.

### Key Highlights

- **Infrastructure**: 3 production systems deployed on Ubuntu VPS with 99.9% uptime
- **Performance**: 40-60% API optimization through query tuning and Redis caching (60%+ improvement)
- **Security**: Enterprise-grade RBAC, encryption, JWT authentication, and audit logging
- **Automation**: CI/CD pipelines via GitHub Actions with SSH key authentication
- **Testing**: Jest-based test suites with 85%+ code coverage

## Development

### Prerequisites

- Node.js 18+
- npm

### Installation

```bash
npm install
```

### Scripts

```bash
npm run dev      # Start development server on port 3000
npm run build    # Create production build
npm run preview  # Preview production build
npm run lint     # Run TypeScript type checking
```
